1. Задание 1: Основные переменные.

Файл greetings.py

Созданы базовые переменные для заметки: username, title, content, status, created_date, issue_date.
Выполнен вывод значений через print.

2. Задание 2: Изменение формата вывода даты.

Файл date_changer.py

Реализовано преобразование формата отображения дат для пользователя, путём исключения года при выводе переменных created_date и issue_date.

3. Задание 3: Ввод данных через input.

Файл add_input.py

Выполнена инициализация переменных через пользовательский ввод (input).
Реализованы подсказки для пользователя о том, как вводить данные, включая формат дат.

4. Задание 4: Добавление списка заголовков.

Файл add_list.py

Реализован в цикле ввод и сохранение в список трёх заголовков (цикл выбран в качестве решения, поскольку так изящнее). 
Вывод организован с отображением списка заголовков.

5. Задание 5: Использование списка.

Файл  final.py

Все данные организованы в список a_note для заметки, включая:
Поля: username - индекс 0, content - индекс 1, status - индекс 2, created_date - индекс 3, issue_date - индекс 4.
Список заголовков как значение с индексом 5. 
Реализован вывод данных списка на экран в структурированном виде с пояснениями.
